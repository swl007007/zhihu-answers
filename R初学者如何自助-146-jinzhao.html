R初学者如何自助
jinzhao
2017-01-07

初学者在学习R语言过程中都会碰到各种问题，那么，R语言初学者如何自助？在网上有非常多好的资源能够提供帮助。之前我们讲过 知乎专栏 ,文中包括了各种资源，提供具体的学习语言、提问、参与社区讨论的方法，本文接下来讨论如何利用这些资源。

学习R

如果你是R语言小白，那么以下资源都是非常棒的学习起点t:

  • basic R tutorial 深入浅出教你R函数.
  • Introduction to R 提供更加完成详细的R语言概述.
  • introduction to R for those coming from other languages ，与其他语言关系；
  • The R Reference Card 针对性非常强的小卡片
  • 免费R语言与统计学 Introduction to Statistics

如果你非常熟悉R语言，你想更加深入的了解其中的某一部分，请参考

  • Stack Overflow 这个大家都必须知道的 books for learning the R language .
  • Contributed Documentation 社区论坛提供的教程，非常容易理解，针对性强。.
  • 如果你有信心， The R Inferno 提供深度解析R语言（性能方向）
  • Google R Style Guide 扫盲

针对性提问

有问题说明你有进步了，那么 RSeek meta search engine , 其实还是菜鸟，你提过的问题肯定别人也遇到过，并且都已经解决了，所以，学会提问（英语）

如果，以上还没有解决你的问题，那么，来更专业点的

  • R-help mailing list 先搜一下问题，看一下 search the list archives 是否已经有你问题
  • Stack Overflow 非常重要的，不解释了 answers to questions about R .
  • 除了R语言，统计学的问题在这里 CrossValidated Q&A community .

根据需求找包

R语言是解决问题的，所以要针对性的找到包， available from CRAN 有太多的包，如何找到需要的包，可以通过一下的方式：

  • RStartHere , 根据数据科学门类进行分类管理.
  • CRAN Task Views 提供各个包的纲要
  • crantastic 社区论坛，各种review，可以快速了解行业与相应包，都是心得.

R语言相关新闻

R语言社区快速增长，也发生了好多有意思的事情，如果可以通过一下的方式来追踪这些动态:

  • R-bloggers 大牛云集
  • R Videos channel 更容易理解
