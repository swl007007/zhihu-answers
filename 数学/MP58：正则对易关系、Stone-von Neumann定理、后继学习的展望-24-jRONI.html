泛函分析和量子力学：我们学到了什么？


原本计划在介绍完Stone-von Neumann定理后继续讲几个主题：

  • 算子半群
  • 角动量
  • 密度矩阵
  • 二次量子化
  • 路径积分

但是还有几个子系列在等着，实在没有办法照顾到了。我们打算在后面的经典场论、分析力学、量子场论中讨论这些主题，或者用更现代的数学工具来处理这些问题。比如，算子半群属于泛函分析的高级主题，在发展方程中可能会继续讨论；角动量源于经典分析力学的主题，未来可以用更近代的几何思想（Lie群、辛几何）讨论；路径积分可以在经典场论中讨论，也可以在量子场论讨论，其数学的严密性需要算子半群、Trotter乘积等方面的概念。


我们差不多用了二十讲介绍泛函分析与量子力学。对读者的知识要求，从物理上可以说只需要中学所具备的原子物理的一点现象描述的定性知识，以及数学上最基本的线性代数。我们究竟学到了什么呢？现在，请回头再看：

  • Sakurai: Modern Quantum Mechanics
  • Landau: 非相对论量子力学
  • 顾樵: 量子力学（推荐）
  • 以及曾谨言的量子力学（不推荐，看目录就可以了）

将会发现，标准的讲法和我们的系列差别实在是太大了。我们压根就没有打算按照传统的方法讲，如果是那样就没有动力写这个专栏了。然而，如果读者已经浏览过本系列中的大部分内容，再去看这些标准的量子力学，会感觉非常的轻松——更多的物理现象、更多的计算，有了基础以后一切不至于不知所云。

实际上我们在MP40~44中完成了量子力学建模的问题，在MP45~47中解了最简单的Schrödinger方程，对于量子力学是什么，差不多有了个印象。我们用长达9讲来讨论线性算子，这自然是比物理系的培养方案要强的内功了，毕竟我们的系列还挂着泛函分析的名头。按照国内比较好的教材：

  • 张恭庆: 泛函分析（上下册）

上册的泛函分析是给数学系本科生的主课教材，下册是给数学系研究生的通科教材。我们的线性算子主题大部分的内容是比较靠后的内容。所以，没有接触过泛函分析的读者如果回头再去看泛函分析，会惊奇地发现我们竟然跳过了泛函分析最重要的一些定理，如开映像定理、闭图像定理、共鸣定理、Hahn-Banach定理，而且我们几乎没有怎么讲完备性和Bananch空间，就直接上手玩Hilbert空间上的线性算子了。反过来说，我们学到的一些东西，不仅物理系的同学闻所未闻，可能数学系的本科生也是一头雾水。这样真的好么？没关系的，从物理上说，毕竟许多人都认为量子力学的难是在于线性代数的基础理解不到位，而不是缺乏泛函分析的知识，而从我们的角度看，介绍更多的高级/近代数学思想，是用来降维打击的。从数学上手，大量上手使用线性算子理论，同时了解一些算子代数和算子半群的思想，可以反复地加强数学的理解，再回头去补基础也是事半功倍的。


从研究的角度，完成了现在的泛函分析与量子力学，我们足以往后继续推进后面的系列了。真正的研究都是缺什么补什么，带着问题学知识。我们系列的目的是尽快看到问题（我们的责任），再往回补（读者自己的事，不归我们管）。回补的参考书，除了上面几本（再次排除曾谨言的量子力学），还有：

  • Reed & Simon. Methods of Modern Mathematical Physics I: Functional Analysis; // 主要是第一册的算子和谱分析部分，后面的三本也可以用来作为深入学习。
  • Lax. Functional Analysis; // 这本书是一章理论一章实例的节奏，适合希望系统学习泛函分析的读者
  • Yosida. Functional Analysis; // 起点很高，一气呵成，适合泛函分析的职业玩家
  • Halmos. Measure Theory; // 希望补测度和实分析方面的基础，可以参考这本
  • Hall. Quantum Theory for Mathematicians; // 希望同时在数学和物理上都照顾到的，建议把这本书系统地读下来，并且，它是GTM 267，读者有了本系列的基础，会发现去读一本GTM是如此畅快的事
